核心实现亮点

Axios 请求封装 - 自动 Token 注入与 401 处理

请求拦截器自动从 localStorage 获取 JWT Token 并添加 Authorization 头；响应拦截器统一处理错误，401 状态码自动清除 Token 并跳转登录页；FormData 上传时自动移除 Content-Type 让浏览器设置 boundary。

Socket.io 客户端服务 - 实时通知与在线状态

封装 socketService 单例，支持 connect/disconnect/reconnect；自动监听 notification:new 事件并更新 notificationStore；通过 join:projects/leave:project 管理项目房间；浏览器 Notification API 桌面推送。

路由守卫 - 登录状态拦截

全局前置守卫检查 Token，未登录重定向至登录页；登录后自动重定向到原目标页面；支持路由懒加载优化首屏性能。

国际化 i18n - 中英文切换

基于 vue-i18n，zh.ts / en.ts 各 ~380 行翻译；覆盖所有页面文本与组件文案；支持动态语言切换无需刷新。

文件系统 - 类文件管理器实现

fileStore (~23KB) 实现完整的文件夹树遍历、面包屑导航、拖拽移动；支持软删除与回收站还原；Multer 文件上传 + 下载链接生成。

Dashboard 看板 - 数据可视化

集成 ECharts 展示项目进度环形图、任务状态柱状图、团队绩效折线图；里程碑时间线展示；活动日志实时更新。

在线状态存储

🔌 WebSocket 连接

客户端登录后自动建立 Socket.io 连接，JWT Token 作为握手认证凭据。支持 websocket 和 polling 两种传输方式自动降级。

🔴 Redis 在线管理

用户连接后以 user_online:{userId} 为键存储 Socket ID，TTL 300秒。断开连接时自动清除。通知推送前先检查 Redis 判断接收者是否在线。

💬 房间机制

用户登录后加入 user:{userId} 个人房间；进入项目时加入 project:{projectId} 项目房间，实现精准的消息推送目标。

🔔 通知推送流

创建通知 → 写入 PostgreSQL → 检查 Redis 在线状态 → 在线则 io.to() 实时推送 → 客户端更新 Store + 浏览器 Notification
